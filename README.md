JAEL AI es un proyecto que muestra un sitio orientado a la creación de campañas publicitarias asistidas por IA y un dashboard de rendimiento que visualiza métricas clave (CPA, CTR, distribución del presupuesto, histórico de inversión y eventos de conversión). Además incluye una sección de Contáctanos para captación de leads. Está pensado como un prototipo interactivo que consume datos en formato JSON y los actualiza dinámicamente con AJAX.
